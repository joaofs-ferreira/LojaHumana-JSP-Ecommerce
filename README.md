# LojaHumana-JSP-Ecommerce

> Academic e-commerce web application built with Java JSP, MySQL and Apache Tomcat.

---

## Description
**LojaHumana** is a lightweight e-commerce web application developed in Java using JSP. The project allows users to browse products organized by categories and simulate a shopping experience through a virtual cart.

This project was developed for academic purposes to consolidate knowledge in **Java web development** and **database integration**.

---

## Features
* **Category Management:** Organize and browse products by specific groups.
* **Product Management:** Detailed listing and handling of items.
* **Product Filtering:** View products filtered by their respective categories.
* **Shopping Cart:** Add products to a cart and manage selections.
* **Cart Management:** Remove unwanted items before checkout.
* **Order Summary:** View a complete breakdown of the items ready for purchase.

---

## Tech Stack

### Backend
* **Java**: Core programming language.
* **JSP (JavaServer Pages)**: For dynamic content rendering.
* **JDBC**: For database connectivity and persistence.

### Frontend
* **HTML5 / CSS3**: Structure and styling.
* **JavaScript**: Client-side interactivity.

### Database & Server
* **MySQL**: Relational database management.
* **Apache Tomcat**: Web server and servlet container.
